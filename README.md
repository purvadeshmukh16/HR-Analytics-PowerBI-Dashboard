# HR Analytics Dashboard using Power BI

## Project Overview
This project focuses on analyzing employee data to identify key factors affecting employee attrition and performance.  
The dashboard is designed to help HR teams make data-driven decisions.

##  Tools Used
- Power BI
- DAX
- Excel / CSV Dataset

##  Dashboard Pages
### 1. HR Overview
- Total Employees
- Attrition Count & Rate
- Attrition by Department, Age Group & Salary Band

### 2. Employee Attrition Analysis
- Attrition by Job Role
- Attrition Trend by Experience
- Interactive slicers for filtering

### 3. Employee Performance Insights
- Performance Rating Distribution
- Salary vs Performance
- Average Performance Rating & Salary

##  Key Insights
- Higher attrition observed in specific job roles
- Early years of experience show higher attrition
- Performance ratings are fairly consistent across departments
